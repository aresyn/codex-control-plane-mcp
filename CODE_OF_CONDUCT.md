# Code of Conduct

## Our Pledge

We want this project to be a useful, respectful place for people building local
agent tooling. Be direct, technical, and considerate.

## Expected Behavior

- Use welcoming and inclusive language.
- Respect different experience levels and operating environments.
- Keep feedback focused on code, behavior, documentation, and reproducibility.
- Avoid sharing private prompts, logs, credentials, or personal data.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing private information without explicit permission.
- Deliberately derailing technical discussion.
- Submitting code or examples that exfiltrate secrets or mutate user data without clear consent.

## Enforcement

Project maintainers may edit, hide, or remove comments, issues, pull requests,
or other contributions that violate this code of conduct.
